# Console Bank Application

## Purpose
A pure Java console-based banking application designed to demonstrate clean code principles, package-based architecture, and object-oriented design patterns.

## Architecture
- **Base Package**: `com.kesavan.bank`
- **Modules**:
  - `app`: Application entry point and coordination.
  - `model`: Data structures representing banking entities.
  - `service`: Business logic and operations.
  - `exception`: Application-specific error handling.
  - `util`: Common utilities and helpers.

## Requirements
- Java 11 or higher.
